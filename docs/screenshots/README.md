# Скриншоты для README

Добавьте сюда файлы перед публикацией репозитория:

| Файл | Что снять |
|------|-----------|
| `landing.png` | Главная страница (Hero + секции) |
| `shop.png` | Страница `/game` — магазин цветов |
| `garden.png` | Страница `/game` — мой сад с цветами |
| `profile.png` | Блок профиля с подключённым кошельком |
| `gameplay.gif` | Запись: подключение кошелька → покупка → сад |

## Как сделать GIF (macOS)

1. Запустите `npm start`
2. Запишите экран через QuickTime или [Kap](https://getkap.co/)
3. Конвертируйте в GIF при необходимости (ezgif.com, ffmpeg)

## Рекомендуемый размер

- PNG: ширина 1280px
- GIF: до 15 секунд, ≤ 5 MB
